Renders a Lucide stroke icon. Use across the product for navigation, buttons, and status. The page must load the Lucide UMD script (`window.lucide`). Stroke weight is a refined 1.75 by default to match the premium tone.

```jsx
<Icon name="shield" size={20} />
<Icon name="landmark" size={24} color="var(--color-primary)" />
```

Common names in EverLedge: `shield, home, users, archive, file-text, landmark, calendar, folder, gift, trending-up, bell, settings, chevron-right, check, plus`.
