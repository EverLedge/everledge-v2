The fundamental container of EverLedge — clean white, hairline border, soft cool shadow, generous padding. Set `interactive` for clickable cards (adds a gentle 2px lift on hover). Use `accent="gold"` sparingly to denote a premium feature or estate milestone.

```jsx
<Card>
  <h3 className="el-h3">Net Estate Value</h3>
  <p className="el-body">Your affairs, organised.</p>
</Card>

<Card interactive accent="gold">…</Card>
```
